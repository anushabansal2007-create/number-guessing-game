# Future Improvements

The project can be improved with the following features:

1. **Difficulty Levels**
   - Easy mode (1–50)
   - Medium mode (1–100)
   - Hard mode (1–500)

2. **Limited Attempts**
   - Player loses after incorrect guesses exceed a limit.

3. **Score System**
   - Reward players for fewer attempts.

4. **Graphical User Interface (GUI)**
   - Build UI using Tkinter or PyQt.

5. **Play Again Option**
   - Allow user to restart the game without closing the program.

6. **Sound & Color Output**
   - Add colorful text or sound notifications for correct guesses.
