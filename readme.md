This mini project is a Number Guessing Game built using Python.
The computer randomly selects a number, and the player has to guess it.
After each guess, the program gives hints like Too High, Too Low, or Correct, along with the total number of attempts.
Technologies Used
Python 3
Visual Studio Code
Git & GitHub

How to Run the Game
Install Python 3 on your system
Open the project folder in VS Code
Open the terminal
Run this command:
python game.py

Features
Random number generation
Gives hints (Too High / Too Low)
Counts number of attempts
Simple and user-friendly
Console-based interactive game
